# Trademark Filter (V2 高级审美版) — 商标智能筛选工具

## 核心机制：告别“土味词汇堆砌”

本技能已经过迭代，舍弃了老旧的“词库堆砌”算法，引入了现代品牌美学评分引擎。

### “一打三扶”评分引擎：
1. **降维打击（反土味惩罚）**：
   重罚 80-90 年代强微商感/村镇日化感的字，如“娇、媚、艳、芬、芳、莎、妮、妃、黛”，一旦包含两个以上，直接触发负分机制。
2. **扶持流派 A（东方极简）**：
   “观、寻、拾、初、本、素、朴、归、落、兮、水”。类似逐本、观夏，给予最高权重加分。
3. **扶持流派 B（院线科技）**：
   “研、因、态、极、零、玑、科、理、医、源、秘”。类似肌研、修丽可，增加专业信赖感。
4. **扶持流派 C（现代 Chic）**：
   “光、悦、己、印、纪、谜、彩”。增加现代女性的独立前卫感。

## 触发条件
- 有大批爬取的商标数据（JSON 格式）
- 用户强调：高级、大气、不俗气、现代感、好记的名字
- 面临新品牌注册的选名环节

## 执行流程

### Step 1: 运行核心引擎
使用已沉淀的深度审美脚本：
```bash
.venv/bin/python3 scripts/trademark_filter.py
```
*(注：该脚本内置了对极简字体的星标（`★`）打印功能和反土味负分结算体系）*

### Step 2: 审视输出与调整
输出会自带 `意境流派` 分数与 `反土味惩罚` 分数。
- 如果某行业需要特定审美（如食品行业需要偏向“鲜、纯、源”），可以直接在脚本的 `ORIENTAL_CHARS` 或 `LAB_CHARS` 中拓展同义字。

### Step 3: 产物交付
- 脚本自动生成高度格式化的极简黑白配色的 Excel 排行榜（包含意境打分条目）。

在 `biaobiaoku-scraper` 10,000 条日化商标筛选中：

- 挖掘出现代名字：“兮密码”、“语如”、“露本集”、“溪白泉”等高质量候选。
